# TerraWeek Day 05 — Terraform Modules 🧩

Day 5 of my #TerraWeek challenge focused on one of Terraform's most powerful features: **Modules**.

Instead of writing infrastructure from scratch every time, I learned how to package Terraform configurations into reusable, versioned building blocks.

## 📚 What I Learned

- Creating reusable Terraform modules
- Root modules vs Child modules
- Module inputs and outputs
- Creating multiple resources with `for_each`
- Input validation
- Publishing modules to GitHub
- Using modules via `git::` sources and Git tags
- Module composition

## 🛠️ Technologies Used

- Terraform
- AWS
- EC2
- Git & GitHub
